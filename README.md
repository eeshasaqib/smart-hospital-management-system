# Smart Hospital Management System

## Project Overview

The Smart Hospital Management System is a comprehensive Database Management System developed using Oracle SQL and PL/SQL. The system automates hospital operations including patient management, doctor assignments, appointments, treatments, billing, prescriptions, medical records, laboratory tests, and ward management.

---

## Features

- Patient Management
- Doctor Management
- Nurse Management
- Receptionist Management
- Department and Ward Management
- Appointment Scheduling
- Treatment Tracking
- Billing and Payments
- Medical Records
- Prescriptions
- Laboratory Tests
- Role-Based Security
- Views for Data Abstraction
- Triggers for Business Rules
- PL/SQL Procedures for Hospital Workflows
- Indexing for Query Optimization
